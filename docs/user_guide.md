# Chartici User Guide

Welcome to Chartici. This guide explains the available layout engines and diagram structures you can use when generating or editing charts.

### ✦ Flowchart
*logical step-by-step processes or algorithms.*

**Editor Specifics:**
- **Tools**: You can use **process**, **circle**, **oval**, **rhombus**, **text** elements.

---

### ✦ Sequence
*chronological interactions between systems or actors.*

**Editor Specifics:**
- **Tools**: You can use **process**, **circle**, **text** elements.

---

### ✦ Entity-Relationship
*database schemas, entities, and relationships.*

**Editor Specifics:**
- **Tools**: You can use **process**, **text** elements.

---

### ✦ Radial
*mind-maps, concentric layers, or hub-and-spoke architectures.*

**Editor Specifics:**
- **Tools**: You can use **process**, **circle**, **text** elements.

---

### ✦ Array
*memory buffers, queues, or sequential data structures.*

**Editor Specifics:**
- **Tools**: You can use **process**, **text** elements.

---

### ✦ Matrix
*grid-like comparisons, or categorization into distinct cluster zones/cells.*

**Editor Specifics:**
- **Tools**: You can use **process**, **text** elements.

---

### ✦ Timeline
*events plotted on a generic chronological spine.*

**Editor Specifics:**
- **Tools**: You can use **chevron**, **process**, **circle**, **text** elements.

---

### ✦ Tree
*strict hierarchical org-charts or breakdowns.*

**Editor Specifics:**
- **Tools**: You can use **process**, **circle**, **text** elements.

---

### ✦ Pie Chart
*breakdown of items into proportional circular slices.*

**Editor Specifics:**
- **Tools**: You can use **pie_slice** elements.
- **Connections**: Connections (Edges) are disabled for this type.
- **Values**: Requires numerical values (available as a `VALUE` input inside the left sidebar).

---
